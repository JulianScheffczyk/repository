MoveClassify
Diese Website wurde mit Streamlit erstellt und ermöglicht es Benutzern, anhand von einer JSON File ihre Bewegung zu klassifizieren und gibt eine durchschnittliche Kalorienberechnung aus.

Anforderungen
Sensor Logger App

Verwendung
Nehmen sie Bewegungsdateien mit der Sensor Logger App auf mit den folgenden Sensoren:
Accelerometer, Gyroscope, Orientation (jeweils 1Hz sample rate)
Exportieren in einen JSON File
